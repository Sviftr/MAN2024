Історичні карти та старі зображення:

Каталог містить сервіси для картографії (OpenStreetMap), супутникових даних (Sentinel Hub), моніторингу транспорту (MarineTraffic, Flightradar24), погоди (Ventusky) та ігор (GeoGuessr), а також спеціалізовані карти (Submarine Cable Map, Starlink map).

Картографічні сервіси та супутникові дані:

Каталог містить сервіси для роботи з історичними картами та старими зображеннями. Вони дозволяють досліджувати карти минулих епох, аналізувати зміни місцевості, вивчати архівні супутникові дані та зображення для історичних і наукових досліджень.

Моніторинг та аналітика:

Каталог містить сервіси для моніторингу та аналітики. Вони забезпечують спостереження за погодою, рухом транспорту, супутниками, природними явищами, а також інструменти для аналізу геоданих і створення інтерактивних візуалізацій.

Рослинність та біорізноманіття:

Каталог містить сервіси для дослідження рослинності та біорізноманіття. Вони допомагають аналізувати екосистеми, моніторити зміни в ландшафтах, вивчати флору і фауну за допомогою супутникових даних і інтерактивних карт.

OSINT (Open Source Intelligence):

Каталог містить сервіси для OSINT (розвідки з відкритих джерел). Вони забезпечують збір, аналіз і візуалізацію відкритих даних, включаючи моніторинг транспорту, супутників, погодних умов, геолокацій та інших джерел для інформаційних досліджень.

Інші корисні сервіси:

Каталог містить інші корисні сервіси, що охоплюють різноманітні завдання: від аналізу геоданих і інтерактивних карт до спеціалізованих інструментів для дослідження інфраструктури, інтернет-покриття чи стандартів електромереж.
