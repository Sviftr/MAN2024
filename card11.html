UHMI

Онлайн-інструмент для моніторингу екологічних даних та моніторингу природних ресурсів. Для заміни корисну інформацію для аналізу стану середовища.

maps.uhmi.org.ua

EcoWater

Платформа для моніторингу якості води в Україні, з можливістю уточнення даних екологічної сталі з водою.

http://monitoring.davr.gov.ua/EcoWaterMon/GDKMap/Index

Глобальна лісова варта

Інструмент для моніторингу лісів у режимі реального часу, завдяки чому ми можемо виконувати буріння лісів та зміни в екосистемах.

https://www.globalforestwatch.org

Якість повітря

Сервіс для якості світу. Крім того, це допомагає змінювати рівень контракту в реальному часі.

https://air-quality.com

OneSoil

Інтерактивна карта з полями та посівами, які були виявлені за допомогою ІІ. Дозволяє отримати уявлення про місцеві та світові тенденції в рослинництві та вивчити розподіл урожаю та посівні площі.

https://map.onesoil.ai/2023/#2.3/44.35/-43.66

OCEARCH

Онлайн-платформа для спостереження за пересуванням великих морських істот, акул, зомбі та китів.

https://www.ocearch.org/tracker

Динамічний світ

Інтерактивний ресурс для розвитку змії на планеті, з метою надання даних про різні екологічні аспекти за допомогою аналізу супутникових знімків.

ПЕРЕЙТИ

WAQI

Платформа для моніторингу якості війни в різних точках світу, що є актуальним для розвитку світу.

https://waqi.info/#/c/4.722/7.667/2.4z

ELECTRICITYMAPS

Онлайн-інструмент для контролю за створенням електроенергії, ін'єкцією в навколишнє середовище та джерелом енергії в режимі реального часу.

https://app.electricitymaps.com/map/24h
